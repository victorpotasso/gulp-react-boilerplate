## Installation

```
npm install
```
## NPM run scripts

* `npm run start`
* `npm run serve`
* `npm run build`
* `npm run test`
* `npm run generate`
* `npm run storybook`
* `npm run build-storybook`

## Features

* BabelJS
* React
* Redux
* Jest/Enzyme
* Storybook
* Browsersync
* Generators
